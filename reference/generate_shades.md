# Generate shades of a colour

Generate shades of a colour

## Usage

``` r
generate_shades(color, n, min_alpha = 0.3, max_alpha = 1)
```

## Arguments

- color:

  Base colour

- n:

  Number of shades

- min_alpha:

  Minimum alpha (transparency)

- max_alpha:

  Maximum alpha (transparency)

## Value

Character vector of colours
