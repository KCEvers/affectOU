# Print simulation object

Display overview of an `simulate_affectOU` object without printing the
full data array.

## Usage

``` r
# S3 method for class 'simulate_affectOU'
print(x, digits = 3, ...)
```

## Arguments

- x:

  A `simulate_affectOU` object

- digits:

  Number of digits for numeric display

- ...:

  Additional arguments (unused)
