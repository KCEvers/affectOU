# Set up axes for a plot panel

Set up axes for a plot panel

## Usage

``` r
setup_panel(xlim, ylim, P, main = NULL, xlab = NULL, ylab = NULL)
```

## Arguments

- xlim:

  X-axis limits

- ylim:

  Y-axis limits

- P:

  Parameter list

- main:

  Main title (NULL to suppress)

- xlab:

  X-axis label (NULL to suppress)

- ylab:

  Y-axis label (NULL to suppress)

## Value

None (invisible)
